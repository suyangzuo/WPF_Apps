# WPF 应用集合

## 项目简介

WPF应用集合是一个基于.NET框架和Windows Presentation Foundation (WPF)技术开发的桌面应用程序集合。该解决方案包含多个独立的WPF应用程序，每个应用程序都展示了WPF框架的不同特性和功能。

## 解决方案结构

```
WPF_Apps/
├── WPF_Clock/                # 桌面时钟应用
│   ├── App.xaml              # 应用程序资源和全局样式定义
│   ├── App.xaml.cs           # 应用程序入口点和初始化逻辑
│   ├── MainWindow.xaml       # 主窗口界面设计和布局
│   ├── MainWindow.xaml.cs    # 主窗口功能和交互逻辑
│   ├── AssemblyInfo.cs       # 程序集信息
│   ├── WPF_Clock.csproj      # 项目配置文件
│   ├── README.md             # 项目详细文档
│   └── Assets/               # 资源文件夹
│       └── ICO/
│           └── Clock.ico     # 应用程序图标
├── WPF_应用集.slnx           # 解决方案文件
└── README.md                 # 本文件，解决方案总览
```

## 包含的应用

### 1. WPF_Clock - 桌面时钟应用

一个功能丰富的桌面时钟应用，提供以下特性：

- **双重时钟显示**：同时提供数字时钟和模拟时钟显示
- **深色主题**：完整的深色主题界面，提供舒适的视觉体验
- **自定义窗口样式**：无边框设计，圆角边框，自定义标题栏
- **响应式交互**：按钮悬停效果，平滑的视觉过渡
- **可配置选项**：通过菜单控制时钟显示

**技术亮点**：
- 使用DispatcherTimer实现精确的时间更新
- 通过数学计算实现时钟指针的精确定位
- 完全自定义的窗口样式和控件模板
- 资源管理和深色主题实现

## 技术栈

- **.NET 10**：使用最新的.NET框架
- **WPF (Windows Presentation Foundation)**：用于构建用户界面
- **XAML**：用于界面设计和布局
- **C#**：用于实现应用逻辑和功能

## 系统要求

- **操作系统**：Windows 10/11 或更高版本
- **运行时**：.NET 10 运行时环境
- **开发环境**：Visual Studio 2022 或更高版本

## 构建与运行

### 构建整个解决方案

1. 使用Visual Studio打开`WPF_应用集.slnx`解决方案文件
2. 选择Release配置
3. 点击"生成"->"生成解决方案"

### 运行单个应用

1. 在解决方案资源管理器中，右键点击要运行的项目
2. 选择"设为启动项目"
3. 按F5或点击"启动调试"按钮运行

### 发布应用

每个项目都配置了单文件发布，可以使用以下命令发布：

```bash
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true
```

## 开发指南

### 添加新应用

1. 在解决方案文件夹中创建新的项目文件夹
2. 在Visual Studio中添加新的WPF项目
3. 更新解决方案文件，添加新项目引用

### 代码规范

- 遵循C#编码规范
- 为所有公共成员添加XML文档注释
- 使用有意义的变量和方法命名
- 保持代码整洁和模块化

### 资源管理

- 将图像和图标等资源放在项目的Assets文件夹下
- 使用相对路径引用资源
- 为不同分辨率提供适当的资源
